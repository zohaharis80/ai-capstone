# Frontend AI Engineering Capstone

This repository contains my capstone project for the FlyRank Frontend AI Engineering track.

## Project Overview

This capstone focuses on building a production-ready frontend application with React and modern JavaScript. The project demonstrates AI-assisted development workflows using Cursor, with an emphasis on clean component architecture, responsive UI, and maintainable code.
